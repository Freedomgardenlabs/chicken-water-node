# Chicken Water Node – V1

Autonomous water distribution node for backyard chickens.

---

## Purpose

This system automates water circulation and prepares infrastructure for future integration into a modular agriculture control architecture (ADAM).

Designed for:

- Backyard chicken owners
- Small-scale micro-farms
- Autonomous agriculture experiments

---

## System Architecture (V1)

- Arduino-based controller
- 5V submersible pump
- Optocoupler isolated relay (LOW trigger)
- External power supply
- Manual timing logic

---

## Development Roadmap

### V1
Basic autonomous pump control.

### V1.1
Solar panel integration

Light sensor logic  

### V2
Water level monitoring

Central control communication  

---

## Long-Term Goal

This node will operate as an independent module inside a distributed agriculture system managed by ADAM.
